# Lite Backend Template

A modern Hono + Drizzle ORM + Bun backend template with decorator-driven architecture.

## Features

- 🚀 **Hono** - Fast, lightweight web framework
- 🗄️ **Drizzle ORM** - Type-safe PostgreSQL ORM
- 📝 **Swagger/OpenAPI** - Auto-generated from decorators
- ✅ **Validation** - class-validator integration
- 🎨 **Decorators** - Express-like decorators for routing
- 📦 **Repository Pattern** - Clean architecture

## Usage

```bash
# Create a new project
bunx github:DoniLite/Lite-Backend-Template my-project
```
